# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## Framework & Library

↪ React TS 18+

↪ Tailwind CSS

↪ Framer Motion

↪ AOS - Animate on scroll library

↪ Code Splitting

↪ Google Fonts

↪ Easy Folder Structure

↪ Organized Code Structure
